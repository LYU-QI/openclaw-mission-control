# 测试计划：飞书同步 + Mission 自动下发全链路测试

## 项目信息
- **项目名称**: OpenClaw Mission Control - 飞书同步自动化
- **版本**: v1.0
- **测试类型**: E2E 集成测试
- **测试日期**: 2026-03-15
- **测试人员**: Claude Code

## 背景

本次测试覆盖飞书多维表格同步到 Mission Control 并自动下发 Mission 给 OpenClaw 执行的完整链路。

## 测试范围

### 功能模块
1. **飞书同步模块**
   - 增量同步（新增/更新/跳过）
   - 字段映射
   - 看板映射
   - 冲突检测

2. **自动下发模块**
   - auto_dispatch 配置
   - 新任务自动创建 Mission
   - Mission 自动下发执行

3. **结果回写模块**
   - Mission 完成自动回写飞书
   - 状态、结果、风险字段更新

### 测试目标
1. 验证飞书同步功能正常工作
2. 验证 auto_dispatch 自动下发 Mission 功能
3. 验证任务状态流转正确
4. 验证结果回写飞书功能
5. 验证唯一约束防止重复任务

## 测试策略

### 测试环境
- **后端**: http://127.0.0.1:8000
- **数据库**: PostgreSQL (Docker)
- **飞书 API**: Real API (生产环境)
- **认证**: Local Auth Token

### 测试方法
- API 手动调用测试
- 数据库状态验证
- 响应结果验证
